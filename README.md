Про программу на джаве - открыть файл cart_program.java:
Есть 2 класса - Product и Cart и у каждого свои методы. С их использованием в Main выполняется задание из 3-го пункта файла Word - добавляются и удаляются продукты.
При запуске программы сразу выводится то, что требуется в задании.



Про работу с гит: Метод реализован на питоне.
Введите числа через пробел и вам выведется максималоьный элемент.
Программа перебирает разделённый список по пробелам и превращает
строки в числа. Затем с помощью метода max ищет максимум.
UPDATE теперь пользователь может вводить количество чисел.
